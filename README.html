# ecommerce_api

Evaluating API

Including new product:

At Postman:

POST http://localhost:8080/product

header > Content-Type: application/json

use request.body to mount the product

body >

{

"id": "unique id",

"name": "product name",

"total_stock": number int,

"passing_stock": number int,

"price_from": number float,

"price_for": number float

}

example:

{

"id": "6",

"name": "Djacket",

"total_stock": 10,

"passing_stock": 5,

"price_from": 9.99,

"price_for": 5.00

}

Search for a product by its id

At Postman:

GET http://localhost:8080/product/:id

header > Content-Type: application/json

example http://localhost:8080/product/6

View all products at db

At Postman:

GET http://localhost:8080/products

header > Content-Type: application/json

example http://localhost:8080/products

Update a specific product from db:

At Postman:

PUT http://localhost:8080/product/:id

header > Content-Type: application/json

use request.body to mount the product as if it is new

body >

{

"id": "new_unique id",

"name": "new_product name",

"total_stock": number int,

"passing_stock": number int,

"price_from": number float,

"price_for": number float

}

example:

{

"id": "7",

"name": "Djacket_2",

"total_stock": 11,

"passing_stock": 5,

"price_from": 9.99,

"price_for": 5.00

}

Obs: Its possible to edit the old id, but its protected to continues been unique

Delete permanently a product from db

At Postman:

DELETE http://localhost:8080/product/:id

example:

DELETE http://localhost:8080/product/7
